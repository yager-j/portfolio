Toy-swapping app for a sustainability tech startup

I architected and built a native iOS app MVP for iPhone and iPad for a sustainability tech startup in Malmö.
The app uses animations, QR code scanning, a map function and Firebase backend to store users' catalogued toys.

SwiftUI – VisionKit – Firebase – ARKit – Async/Await

The user takes a photo of their toys, the app scans the image and extracts each toy from the image using VisionKit. A pixellated filter is applied to the toy images and each toy is added to the user's toy catalog via the Firebase backend.

Users access the catalog of their toys in the app and set up toy swaps with other users. They can swap toys by scanning each other's QR codes. This moves the swapped toys between users' toy catalogues via the Firebase backend.

Users use the app's map features to find other users to swap toys with and set up swaps with other users.

I experimented with AR experiences and in-app minigames. These features are not part of the current MVP version but will be developed further in future iterations of the app.
