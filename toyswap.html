Toy-swapping app for a sustainability tech startup

I architected and built a native iOS app MVP for iPhone and iPad for a sustainability tech startup in Malmö.
The app includes Spline animations, subject lifting, QR code scanning, a map function and Firebase backend to store users' catalogued toys.

SwiftUI – VisionKit – Firebase – ARKit – Async/Await

In the app, the user takes a photo of their toys, the app scans the image and extracts each toy from the image. A pixellated filter is applied to the toy images and each toy is added to the user's toy catalogue via the Firebase backend.

Users choose toys to swap and set up swap meet-ups with other users. To swap toys, each user scans the swapped toy's QR code in the app. This moves the swapped toys between users' toy catalogues via the Firebase backend.

The map feature is used to find others to swap toys with.

I also experimented with AR experiences and in-app minigames. These features are not part of the current MVP version but the ideas explored will serve as a starting point for future iterations of the app.
