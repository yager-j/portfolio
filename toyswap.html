Toy-swapping app MVP for a startup

I architected and built a native iOS app MVP for iPhone and iPad, for a startup in Malmö, Sweden.
The app includes Spline animation integrations, subject lifting functionality, QR code scanning and a map function, with user authentication, database and image storage using Firebase.

SwiftUI – VisionKit – Firebase – ARKit – Async/Await

Using iOS camera integration in the app, the user takes a photo of their toys; the app scans the image and extracts each toy using Apple's VisionKit framework, applies a custom pixellation filter to the images and adds them to the user's toy catalogue.

Users select toys to swap and set up swap meet-ups with other users. To swap toys, users scan the QR code of each other's toy, moving the swapped toys between users' toy catalogues.

Users find others to swap toys with using the map.

As the client wanted to make the app entertaining for children we explored possibilities with AR (using ARKit), as well as in-app minigames.
